# PURGE 使用Go重写优化的Python工具集

好多用起来还挺趁手的安全工具是用Python写的，但Python又不方便随地大小执行，所以我参考佢地用Go对着重写，顺便拆分模块为一系列零散独立的二进制可执行文件。

## 工具列表

看[英文版README](README.md)吧。

## 构建方式

### Windows

确保你已经安装了Go 1.23+。

```powershell
make.cmd
```

### Linux和macOS

确保你已经安装了Go 1.23+和Make。

```bash
make
```

### 使用方式

看各个工具的`-h`帮助信息。
